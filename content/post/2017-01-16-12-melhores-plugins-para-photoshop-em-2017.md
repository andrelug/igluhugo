Nesse artigo listamos os melhores plugins disponíveis para o Photoshop em 2017. Essa super ferramenta da Adobe é sensacional e bastante completa, mas os plugins do Photoshop ajudam a ferramenta a ser ainda mais poderosa com menos esforço e, as vezes, com menos conhecimento.

Existe um número enorme de plugins disponíveis online, mas aqui selecionamos apenas os melhores para que você já possa baixar e usar com rapidez. **A lista abaixo é composta de ambos plugins gratuitos e plugins pagos!**

## <a href="http://ink.chrometaphore.com/" target="_blank">1- Ink</a>

<a href="http://ink.chrometaphore.com/" target="_blank"><img class=" wp-image-396 aligncenter" src="/images/uploads/2017/01/plugin_ink_photoshop.jpg" alt="" width="727" height="414" srcset="/images/uploads/2017/01/plugin_ink_photoshop.jpg 977w, /images/uploads/2017/01/plugin_ink_photoshop-300x171.jpg 300w, /images/uploads/2017/01/plugin_ink_photoshop-768x437.jpg 768w" sizes="(max-width: 727px) 100vw, 727px" /></a>

Ink é uma ferramenta para quem quer trabalhar em grupo. As vezes as pessoas para quem você envia os designs não são tão experientes no Photoshop como você. Então esse plugin ajuda a incluir informações contextuais no seu documento. Você pode adicionar algumas especificações essenciais como formatação de texto, estilos das camadas, efeitos e tamanhos com facilidade. Em seguida todas as informações são juntadas em uma pasta para ficarem bem organizadas.

É um bom plugin para designers enviarem designs para programadores, por exemplo.

**Empresa**: <a href="http://chrometaphore.com/" target="_blank">Chrometaphore</a>

**Preço**: Gratuito

**Bom para**: Web designers

## <a href="http://www.painterartist.com/en/product/particleshop/" target="_blank">2- ParticleShop</a>

<a href="http://www.painterartist.com/en/product/particleshop/" target="_blank"><img class=" wp-image-397 aligncenter" src="/images/uploads/2017/01/particleshop_plugin_photoshop.jpg" alt="" width="751" height="436" srcset="/images/uploads/2017/01/particleshop_plugin_photoshop.jpg 875w, /images/uploads/2017/01/particleshop_plugin_photoshop-300x174.jpg 300w, /images/uploads/2017/01/particleshop_plugin_photoshop-768x446.jpg 768w" sizes="(max-width: 751px) 100vw, 751px" /></a>

Para aqueles fãs do Corel Painter, agora você pode usar as tecnologias de pinceis deles também no Photoshop. Os pincéis podem fazer uma grande diferença na sua produtividade como artista digital e esse plugin oferece sua funcionalidade não destrutiva com suporte para camadas no Photoshop. Além disso ele acompanha 11 pincéis como Debris, Fine Art, Fur, Hair, Space e outros.

**Empresa**: <a href="http://www.painterartist.com/en/" target="_blank">Corel</a>

**Preço**: R$ 158,00

**Bom para**: Ilustradores e Fotógrafos

## <a href="https://www.google.com/nikcollection/" target="_blank">3- Google Nik Collection</a>

<a href="https://www.google.com/nikcollection/" target="_blank"><img class=" wp-image-398 aligncenter" src="/images/uploads/2017/01/nik_collection_plugin_photoshop.jpg" alt="" width="765" height="448" srcset="/images/uploads/2017/01/nik_collection_plugin_photoshop.jpg 1025w, /images/uploads/2017/01/nik_collection_plugin_photoshop-300x176.jpg 300w, /images/uploads/2017/01/nik_collection_plugin_photoshop-768x450.jpg 768w, /images/uploads/2017/01/nik_collection_plugin_photoshop-1024x599.jpg 1024w" sizes="(max-width: 765px) 100vw, 765px" /></a>

Entusiastas fotográficos ficaram muito felizes há quase um ano quando a Google comprou o Nik Collection. Antes os plugins não só eram muito bons, mas bem carinhos. Agora todos são gratuitos e disponíveis para melhorar fotos de qualquer um. Esses plugins para o Photoshop são muito versáteis e feitos para automaticamente aplicar efeitos profissionais em suas fotos. Altamente recomendado para quem gosta de fotografia.

**Empresa**: <a href="https://www.google.com/nikcollection/" target="_blank">Google</a>

**Preço**: Gratuito

**Bom para**: Fotógrafos

## <a href="http://www.gettyimages.com/resources/plugins#" target="_blank">4- Getty Images</a>

<a href="http://www.gettyimages.com/resources/plugins#" target="_blank"><img class=" wp-image-399 aligncenter" src="/images/uploads/2017/01/getty_images_plugin_photoshop.jpg" alt="" width="782" height="518" srcset="/images/uploads/2017/01/getty_images_plugin_photoshop.jpg 868w, /images/uploads/2017/01/getty_images_plugin_photoshop-300x199.jpg 300w, /images/uploads/2017/01/getty_images_plugin_photoshop-768x509.jpg 768w" sizes="(max-width: 782px) 100vw, 782px" /></a>

Eventualmente no processo de criação precisamos de acesso a um banco de imagens (_stock photos_) para compor os projetos. Para facilitar esse processo, o site Getty Images fez um plugin que possibilita a pesquisa e download não só para o Photoshop mas também para o IdDesign e Illustrator. Se você possui uma conta no Getty Images, tem a facilidade também de buscar seu histórico de downloads.

**Empresa**: <a href="http://www.gettyimages.com/" target="_blank">Getty Images</a>

**Preço**: Gratuito

**Bom para**: Designers gráficos e web designers

## <a href="http://www.cs-extensions.com/products/contrastica/" target="_blank">5- Fixel Contrastica 2</a>

<a href="http://www.cs-extensions.com/products/contrastica/" target="_blank"><img class=" wp-image-400 aligncenter" src="/images/uploads/2017/01/fixel_plugin_photoshop.jpg" alt="" width="799" height="531" srcset="/images/uploads/2017/01/fixel_plugin_photoshop.jpg 829w, /images/uploads/2017/01/fixel_plugin_photoshop-300x199.jpg 300w, /images/uploads/2017/01/fixel_plugin_photoshop-768x510.jpg 768w" sizes="(max-width: 799px) 100vw, 799px" /></a>

O Fixel Contrastica 2 está disponível tanto para o Photoshop quanto para o programa After Effects. Esse plugin é um intensificador de contraste inteligente. A empresa Fixel Algorithms é reconhecida por produzir algoritmos inteligentes para o processamento de imagem e vídeo. No Photoshop o Fixel é usado como um filtro bem fácil de se alterar para alcançar os efeitos desejados com precisão. Ele afeta tanto o contraste local quanto o global no mesmo tempo, fazendo a manipulação de imagens ser bem mais rápida e objetiva.

**Empresa**: <a href="http://cc-extensions.com/news/2014/07/fixel-algorithms/index.html" target="_blank">Fixel Algorithms</a>

**Preço**: R$ 100,00

**Bom para**: Fotógrafos

## <a href="http://psdcleaner.madebysource.com/" target="_blank">7- PSD Cleaner</a>

<a href="http://psdcleaner.madebysource.com/" target="_blank"><img class=" wp-image-402 aligncenter" src="/images/uploads/2017/01/psdcleaner_plugin_photoshop.jpg" alt="" width="783" height="486" srcset="/images/uploads/2017/01/psdcleaner_plugin_photoshop.jpg 860w, /images/uploads/2017/01/psdcleaner_plugin_photoshop-300x186.jpg 300w, /images/uploads/2017/01/psdcleaner_plugin_photoshop-768x477.jpg 768w" sizes="(max-width: 783px) 100vw, 783px" /></a>

Organizar, alterar e buscar camadas em um projeto grande no Photoshop é uma tarefa hercúlea e muitas vezes chata. Contudo é algo que grande parte dos designers tem que lidar constantemente. O plugin PSD Cleaner então vem para facilitar esse processo todo. Em um simples painel você pode identificar as camadas não nomeadas, recriar efeitos do Photoshop em CSS, deletar camadas em branco, descobrir quais camadas estão para fora do seu canvas e muito mais.

Empresa: <a href="https://madebysource.com/" target="_blank">Source</a>

Preço: US$ 19,99

Bom para: Designers gráficos e web designers

## <a href="http://dandkagency.com/extensions/velositey-v2/" target="_blank">8- Velositey</a>

<a href="http://dandkagency.com/extensions/velositey-v2/" target="_blank"><img class=" wp-image-403 aligncenter" src="/images/uploads/2017/01/velositey_plugin_photoshop.jpg" alt="" width="800" height="406" srcset="/images/uploads/2017/01/velositey_plugin_photoshop.jpg 1126w, /images/uploads/2017/01/velositey_plugin_photoshop-300x152.jpg 300w, /images/uploads/2017/01/velositey_plugin_photoshop-768x389.jpg 768w, /images/uploads/2017/01/velositey_plugin_photoshop-1024x519.jpg 1024w" sizes="(max-width: 800px) 100vw, 800px" /></a>

Para aqueles que querem criar templates de websites com mais facilidade e velocidade, esse plugin do Photoshop é sensacional. Com mais de 60 templates disponíveis gratuitamente (eles apenas pedem um compartilhamento numa rede social), Velositey é uma ferramenta bem interessante para web designers e desenvolvedores.

Você pode simplesmente baixar um dos templates, carregar no Photoshop e alterar para melhor se adaptar ao seu objetivo e website.

**Empresa**: <a href="http://dandkagency.com/" target="_blank">D&K Agency</a>

**Preço**: Gratuito

**Bom para**: Web designers e desenvolvedores

## <a href="http://plugin.subtlepatterns.com/" target="_blank">9- Subtle Patterns</a>

<a href="http://plugin.subtlepatterns.com/" target="_blank"><img class=" wp-image-404 aligncenter" src="/images/uploads/2017/01/subtle_patterns_plugin_photoshop.jpg" alt="" width="814" height="391" srcset="/images/uploads/2017/01/subtle_patterns_plugin_photoshop.jpg 1090w, /images/uploads/2017/01/subtle_patterns_plugin_photoshop-300x144.jpg 300w, /images/uploads/2017/01/subtle_patterns_plugin_photoshop-768x368.jpg 768w, /images/uploads/2017/01/subtle_patterns_plugin_photoshop-1024x491.jpg 1024w" sizes="(max-width: 814px) 100vw, 814px" /></a>

Subtle Patterns já é uma reconhecida biblioteca de texturas e padrões gratuitos. O plugin que eles possuem faz com que você possa carregar toda sua biblioteca diretamente no seu Photoshop em um painel dedicado. Basta clicar no thumbnail e a textura é aplicada a uma camada como um estilo de camada. Bem eficiente e ajuda a economizar alguns minutos ao longo do dia para quem está constantemente mexendo com padrões e texturas.

**Empresa**: <a href="http://www.atlemo.com/" target="_blank">Atle Mo</a>

**Preço**: US$ 17,99

**Bom para**: Designers gráficos e web designers

**Preço**: US$ 49,99
