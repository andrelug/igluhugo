2017 já está por aqui e agora temos uma seleção com os melhores templates e temas gratuitos em HTML para ajudar você baixar e colocar seu website no ar. Os templates em HTML servem para inspiração, para aproveitar um pedaço deles no seu próprio site ou mesmo para os adaptar em algo com a identidade visual do seu projeto.

Não é possível garantir a boa qualidade de todos os templates online, mas os que estou publicando aqui foram selecionados não só pela sua beleza mas também pelas suas fontes e recomendações de qualidade. Então aproveite para fazer o download gratuito deles.

## <a href="http://freebiesbug.com/code-stuff/karmo-free-html-templates-agencies-creatives/" target="_blank">Template Gratuito Karmo</a>

Karmo é um template moderno, flexível e bem estruturado. Ele foi criado por um designer chamado Tansilav Robert e é baseado no framework Bootstrap 4. É bem documentado, possui 4 páginas diferentes e está cheio de componentes interessantes. Eu diria que seu uso pode ser direcionado para empresas criativas e agências de qualquer mercado.

## <a href="https://pixelbuddha.net/freebie/magnet-portfolio-template" target="_blank">Template de Portfólio Magnet</a>

Magnet é um tema em HTML bem bonito, minimalista e com um design criado para mostrar os trabalhos de uma pessoa criativa da melhor forma possível. Ele é fácil de se utilizar, modificar e customizar e é profissionais que trabalham com elementos digitais poderão fazer muito bom uso dele. **Esse tipo de template não aparece muito e serve mais como uma estrutura para mostrar seu próprio trabalho, então é interessante ter boas imagens dos seus projetos para que fique legal.**

Vez ou outra precisamos criar uma landing page para nossos projetos ou para clientes. Então para agilizar o processo esse template é objetivo em criar algo para esses momentos. Especialmente startups e projetos de tecnologia podem aproveitar desse tema para o início da criação da empresa.

## <a href="https://github.com/BuckyMaler/Global" target="_blank">Tema Moderno em HTML e PSD</a>

Diferentemente dos outros templates gratuitos, você poderá fazer o download desse não só dos arquivos HTML, CSS e JavaScript mas também de seus PSD (arquivo do Photoshop). Apesar de ser apenas uma página, o mecanismos de rolagem simula a transição como se fossem slides de um Power Point (ou Keynote) na tela. É bem interessante e servirá bem empresas mais modernas que buscam algo um pouco diferente na web.
